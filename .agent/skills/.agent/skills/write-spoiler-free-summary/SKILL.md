# Write Spoiler-Free Summary

## Goal
Produce a short summary in Spanish that helps someone decide whether to watch the movie while keeping key plot turns secret.

## Instructions
1. Base the summary on trusted metadata or a short overview, not on fan theories or plot breakdowns.
2. Mention the setup, main characters, setting, and central tension only.
3. Keep the text concise, usually 60 to 120 words.
4. Prefer natural Spanish over literal translation.

## Constraints
- Never mention the ending.
- Never reveal twists, betrayals, deaths, or hidden identities.
- Never describe final-act events.
